## Clone of git profile page 


#### Steps to run in local

`node v10.15.0 (with npm 6.4.1)`

Step 1: `$ git clone https://github.com/shivaprasadks/clone-git.git`

Step 2: `$ cd clone-git`

Step 3: `$ npm i` 

Step 4: `$ npm start`
